Montreal Rental
Location Analytics

Data-driven decision support using Power BI

Data Ingestion – Raw and Bronze Layers

This project helps individuals, families, and shared households identify the borough on the Island of Montreal that best matches their rental needs. It consolidates rental prices, safety indicators, commute time, and access to public services into a single decision-support dashboard.

  • View Dashboard

Project Highlights

Structured Data Pipeline

Multi-layer architecture (Bronze → Silver → Gold → Presentation) supporting scalable transformations and reliable reporting.

Safety and Quality Indicators

Location evaluation includes public safety signals (infractions) and access to essential services for day-to-day living.

Weighted Ranking Model

Users can prioritize criteria by relevance, enabling a personalized ranking of boroughs based on individual needs.

Interactive Power BI Dashboard

An interactive report allows filtering, comparison, and a final recommendation of the top 3 locations.
